Research decisions

Two strategy decisions before the next wave of runs

These choices determine which detector anchors the main interpretation and which second model offers the strongest generalization test.

1. Which neuron set should define the main story?

Should the project stay anchored to the 38 neurons at C=1.0 because that is closest to the paper's sparse-intervention framing, or pivot to the 219 positive neurons at C=3.0 because that detector appears to capture more of the real Gemma signal?

The tradeoff is straightforward: C=1.0 buys paper faithfulness and a clean replication headline, while C=3.0 buys better predictive performance and a story less vulnerable to the 4288 artifact.

2. Which second model adds the most value?

If the replication extends beyond Gemma-3-4B, should the next target be Mistral-Small-24B or Llama-3.3-70B?

Mistral is the cleaner engineering continuation and sits close to the paper's best-behaving family. Llama is the stronger external-validity test, but it asks for a much more expensive deployment story.

Planned experiments

Three highest-priority follow-ups

The first closes the evaluator gap across the full sweep. The other two test whether the effect is specific and what mechanism is actually moving.

Extend text-based scoring (~30–60 min, $0)

Build the missing all-α answer-text scorer beyond the committed α=3.0 remap, then recompute the standard curve and population split from that corrected series.

Negative control (~1 h, $0)

Scale 38 random non-H-neurons by the same α values. This is the cleanest test of whether the observed behavior change is specific or just generic activation damage.

Prompt disentangling (~1–2 h, $0)

Run a no-context standard prompt and an aligned-format prompt so answer selection, answer rendering, and generic degradation can be separated more cleanly.
