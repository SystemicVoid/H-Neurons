Process note

The funnel shape tells a story about methodological choices masquerading as data. The 89% consistency filter is aggressive — it discards every question where the model hesitates even once across 10 samples. This selects for questions the model either knows cold or consistently gets wrong, filtering out the messy middle where hallucination is most interesting. The 118 apostrophe failures (3.8%) are not random either — they systematically remove Irish names, possessives, and contractions. Every pipeline stage shapes what the classifier can learn.

On the data leakage

Finding 66.3% train/test overlap was the first surprise. The paper samples train and test independently, which is standard — but with only 1,391 true-label IDs and both splits requesting 1,000, overlap is mathematically inevitable. The reassuring part: the accuracy drop from 77.7% to 76.5% is only 1.2 percentage points, meaning the leakage inflated results modestly, not catastrophically. The signal survived a clean test. This is worth flagging as a general replication concern — papers that don't report overlap statistics may be hiding this kind of quiet inflation.

What this means: Hallucination detection is not a last-layer phenomenon. The neurons that distinguish true from false activations are distributed across the model, with a strong bias toward early layers — consistent with early commitment to an answer, though other interpretations are possible.
What surprised us

Layer 5 alone hosts 4 of 38 H-Neurons — more than any other layer. In a 34-layer model, the "decision to hallucinate" appearing this early is counterintuitive. One interpretation: early MLP layers encode token familiarity — how well the model "recognizes" the entity before deep reasoning begins. A question about O'Neill vs Shakespeare might already register different confidence patterns in layer 5, before attention has had a chance to compose multi-hop reasoning. This aligns with recent residual stream probing work showing factual recall signals emerging in the first third of transformer depth.

The surprise

One neuron to rule them all?

Neuron (Layer 20, Index 4288) received an L1 weight of 12.17 — 1.65x the runner-up. Is this a genuine "hallucination hub" or an L1 regularization artifact?

The question: L1 regularization has a known "winner-take-all" effect — when two features carry similar signals, L1 arbitrarily assigns all weight to one and zeros the other. A 1.65x gap is unusual enough to warrant investigation. We ran six independent analyses to determine the truth.
Forensic investigation

Six tests, one verdict

Each analysis probes neuron 4288's importance from a different angle. Real signal requires at least 4 of 6 supporting evidence.

1. Single-neuron AUC Artifact

Can neuron 4288 alone classify hallucinations? Its standalone AUC is 0.590 — outperformed by L13:N833 (0.703) and L14:N8547 (0.666). Despite the highest L1 weight, it's not the most informative individual neuron.

2. Distribution separation Artifact

Cohen's d = 0.326 (below 0.5 threshold). The runner-up achieves d = 0.477 with better class separation. Neuron 4288 has signal, but it's not exceptional.

3. C-sweep stability Artifact

The most decisive test. Neuron 4288 is completely absent at C ≤ 0.3. It only enters at C = 1.0, drops to rank 5 at C = 3.0, and rank 11 at C = 10.0. Its dominance is C-specific, not fundamental.

4. Per-example contribution Artifact

Despite 1.65x the weight of #2, neuron 4288 is the largest positive contributor for only 7.4% of examples (threshold: 30%). Median absolute contribution: just 2.6% of total score.

5. Leave-one-out ablation Artifact

Zeroing out neuron 4288 drops accuracy by 1.03pp (below 2pp threshold). Keeping only 4288 yields AUC = 0.828 but accuracy = 50.1% — good ranking, no decision power alone.

6. Correlation structure Artifact

Neuron 4288 correlates at r = 0.492 with L26:N1359 (a zero-weight neuron). L1 likely picked 4288 as the "representative" of a correlated feature group.

Methodological reflection

The investigation of neuron 4288 wasn't planned — it emerged from a simple question: "why is one neuron 1.65x the runner-up?" The six-test framework was designed to be unfalsifiable from any single angle. We set thresholds before running the analyses (AUC>0.60, Cohen's d>0.5, ablation>2pp) to avoid post-hoc rationalization. The unanimous 0/6 verdict was the strongest possible outcome — and it points to a systematic issue with L1-based neuron ranking that likely affects other work using this methodology. The C-sweep (Analysis 3) was the most revealing: the fact that C=3.0 yields +4pp accuracy with 6x more neurons suggests the paper's headline "~35 H-Neurons" is a regularization artifact, not a fundamental property of the model.

The puzzle

Same neurons, opposite curves?

The apparent contradiction dissolves when you separate what the model says from how it formats the answer. H-Neuron amplification increases content compliance but degrades format compliance.

Open questions

What do these neurons actually do?

The data rules out simple stories. Here are three competing hypotheses, each with a concrete experiment that could falsify it.

A

Context-credulity neurons

These neurons modulate how much the model treats provided context as ground truth. High α makes it more credulous (accept counterfactual claims) while reducing attention to formatting constraints. The format degradation is a side effect of over-weighting context.

Test it: Run without any context passage. If these neurons still affect format compliance, credulity isn't the mechanism.

B

Competing-instruction neurons

These neurons amplify content-following at the expense of format-following. The model has limited "instruction budget" — scaling these neurons shifts weight toward semantic compliance (answer the counterfactual question) and away from syntactic compliance (prefix with a letter).

Test it: Create a prompt where format and content instructions are aligned (not competing). If format still degrades, the trade-off theory is wrong.

C

Generic degradation

Extreme activation scaling (α=2–3) simply degrades output precision. Format compliance is more fragile than content compliance because a single missing character breaks it. Content is robust to noise; format isn't.

Test it: Scale 38 random non-H-neurons by the same α values. If format degrades similarly, it's not specific to these neurons.

We don't know yet. And that's the interesting part. The data is consistent with all three hypotheses. Each requires a different ~1 hour experiment to falsify. The negative control (Hypothesis C) is the most urgent because it could invalidate the entire line of reasoning.

On negative controls

Neither the paper nor this replication tests whether scaling 38 random non-H-neurons by the same α values would produce a similar compliance trend. Without this negative control, we can't rule out that any perturbation to intermediate activations increases compliance. This is the single most valuable experiment still missing — it would take ~2 hours and $0 to run.

Takeaways

What we learned

1

The signal is real — and causal

76.5% detection accuracy on a disjoint test set. Under both prompt variants, content-level compliance increases with α (when adjusted for format failures). These neurons causally change behavior.

2

Broader than hallucination — but how?

The data is consistent with a compliance role, but the format-compliance dissociation complicates the picture. Amplifying these neurons increases content-following while degrading format-following — which "compliance" story explains both?

3

Effect is concentrated

80–86% of samples are unaffected by α. The headline effect is diluted across a frozen majority. The causal influence acts on the uncertain margin, not on deeply-held beliefs or obvious fabrications.

4

Prompt engineering dwarfs it

Our slope (2.10%/α) means a full α=0→3 sweep yields 6.3pp. Prompt engineering routinely achieves 30–50pp swings. Neuron scaling establishes causation but is a weak practical lever.

The bigger picture

We went from correlation to causation, and from causation to confusion — which is progress. The paper frames these as "hallucination neurons." Our first prompt variant suggested "compliance neurons." The second variant showed that even "compliance" is too simple: these neurons increase content-following while degrading format-following. Three hypotheses remain in play, each testable with a single experiment. The honest answer is that we don't yet know what these neurons do, but we know more precisely what we don't know.

Next steps

Three experiments to disambiguate

Each targets a specific hypothesis. All run on the same consumer GPU with no additional cost.

Negative control (~1h, $0)

Scale 38 random non-H-neurons by the same α values. Tests Hypothesis C (generic degradation). If compliance rises and format degrades similarly for random neurons, the effect isn't specific to H-Neurons. The single most important missing experiment.

No-context format test (~30min, $0)

Run the standard prompt without a context passage, varying α. Tests Hypothesis A (context-credulity). If format still degrades without any context to be credulous about, the mechanism isn't about context weighting.

Aligned-instruction test (~2h, $0)

Create a prompt where format and content instructions point the same direction. Tests Hypothesis B (competing instructions). If format still degrades when there's no competition, the trade-off theory is wrong.
