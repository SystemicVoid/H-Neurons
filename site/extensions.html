Roadmap table

The page tracks only the four branches that have a concrete scientific hook right now

This is deliberately narrower than the full idea bank in the research notes. The current page keeps only the branches that already have a sharp hypothesis, an affordable falsifier, and a direct path back to the Gemma replication story.

Workstream Status Why it matters Cheap falsifier Main dependency
SAE feature decomposition Primary bet Test whether the 38 H-neurons are only a coarse shortlist and whether a compact set of SAE features is the real mechanism handle. Load the layer-13 MLP-output SAE, project the existing hallucination/truthful activation pairs, train the same L1 logistic regression on 16k SAE features instead of 348k neuron CETT values, and compare AUROC. Cheap falsifier: 1-2 hours GPU. Existing activation pairs plus Gemma Scope SAEs for `gemma-3-4b-it`; full pass is still only 2-3 days.
Suppressive neuron investigation Quick falsifier The ignored negative-weight set may be a natural truthfulness circuit, which is a cleaner safety narrative than merely suppressing bad behavior. Check whether the 38 negative-weight neurons remain stable across C in {0.3, 1.0, 3.0, 10.0}. If a core subset survives, run the inverse intervention by amplifying them and measuring whether compliance decreases. Cheap falsifier: 10 min CPU + 3-4 hours GPU. Classifier artifacts already exist; the only real dependency is finding a stable core worth intervening on.
Swing sample characterization Pure analysis The 6.3pp headline effect is really a 45.7% swing inside a 14% sensitive subpopulation. The key question is whether those samples mark weak knowledge or overridden strong knowledge. Compare swing vs always-compliant vs never-compliant samples on confidence at α=1.0, topic domain, context convincingness, and response length. Cheap falsifier: zero GPU, about one day of analysis. Existing per-sample intervention outputs; best interpreted after the text-rescoring cleanup is complete.
Safety/refusal overlap pilot Safety pilot Directly tests whether factual compliance and harmful-request refusal share circuitry, which is the alignment-tax question in its smallest runnable form. Run 20 manually written harmful prompts at α=0.0 and α=1.0. If suppressing H-neurons makes the model more willing to comply, the overlap is real; if refusal is unchanged, the circuits look separable. A small harmful-prompt set and a disciplined readout; no large new benchmark stack needed for the pilot.
Why these four: they each have a different payoff surface. SAE decomposition is the strongest main sprint, suppressive neurons are the fastest causal falsifier, swing-sample analysis is the cheapest interpretive clarification, and the refusal-overlap pilot is the most direct safety check.
Execution order

The order should separate cheap filters from the main sprint

These four branches are not equal in cost or role. Two are quick ways to learn whether the story is narrower or cleaner than it currently looks, one is the main mechanism bet, and one is the first focused safety pilot.

1

Run the near-free filters first

Check negative-weight stability and characterize the swing population before spending multiple days on a heavier mechanism sprint.

2

Use SAE decomposition as the main sprint

If the SAE probe matches or beats the neuron probe, the project upgrades from sparse-neuron bookkeeping to a feature-level mechanism story on the same model.

3

Use refusal overlap as the targeted safety check

This is the smallest experiment that can tell you whether the same circuitry may be doing double duty for factual compliance and safety refusal.

Roadmap discipline

Research roadmaps drift when every promising idea gets treated like the main branch. This page is meant to force the opposite habit: keep one trunk, then make each extension justify why it deserves to become the next branch.

Top workstreams

The four branches each answer a different failure mode in the current story

The point is not to maximize novelty in the abstract. The point is to ask four concrete questions: is the neuron level too coarse, is there an inverse truthfulness handle, what exactly moves in the swing subset, and does any of this touch refusal circuitry?

SAE feature decomposition

Strongest extension because either outcome is publishable. If SAE features work better, the neuron level is too coarse. If they do not, the H-neurons are unusually monosemantic and that is itself surprising.

Suppressive neuron investigation

Best route to turning the story from "suppress bad behavior" into "amplify a native truthfulness circuit," if the negative-weight set proves stable and causal.

Swing sample characterization

Best route to deciding whether H-neurons override appropriate uncertainty or genuine knowledge. That is the difference between a narrower deployment concern and a much stronger safety alarm.

Safety/refusal overlap pilot

Best route to a direct alignment-tax claim. If refusal moves when H-neurons are suppressed, the project stops being only about hallucination and starts touching refusal geometry.

Strongest paper extension
SAE feature decomposition
This is the cleanest bridge from the current replication to the dominant feature-based interpretability paradigm, while staying on the same model and within local hardware.
Fastest direct safety signal
Refusal-overlap pilot
This is the quickest way to learn whether the same local circuitry may be shared between hallucination compliance and harmful-request refusal.
Decision points

The choice is not "which many ideas are interesting" but "which one earns the next sprint"

Right now the page recommends one main sprint and three smaller falsifiers around it. That keeps the roadmap disciplined: the big bet gets room to breathe, and the cheaper branches still sharpen what the big bet would mean.

Main sprint
Layer-13 SAE first
Preferred because it is the strongest extension with the best "interesting either way" structure and still fits the current hardware and artifact base.
Cheap sidecars
Swing + anti-H first pass
Preferred around the main sprint because both are fast, leverage existing data, and determine whether the safety reading becomes narrower or stronger.
Current recommendation: keep the extensions page focused on these four only for now. SAE decomposition is the main sprint. The suppressive-neuron check, swing-sample audit, and refusal-overlap pilot are the short side experiments that make the eventual story harder to fool.
